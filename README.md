# greedy_search_proyect

## Descripción

Este programa sirve para encontrar el número mínimo de estaciones que hacen falta para cubrir todos los estados del centro de Estados Unidos.

## Configuración y instalaciones necesarias

- Se necesita una versión de python 3.11 o superiores.
- Se necesita una versión de matplotlib 3.10.1 o superiores.

En este pograma hemos usado las siguientes dependencias para el desarolla:
